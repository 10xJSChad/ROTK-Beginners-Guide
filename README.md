# ROTK Beginners Guide

Classes
------
ROTK has a total of three classes, the **gatherer**, **blacksmith**, and **engineer**.
<br>
### Gatherer
The gatherer is the only class capable of gathering ore and wood, it does not have any unique crafting recipes.
<br>
### Blacksmith
